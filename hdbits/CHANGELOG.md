# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and
this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024

### Added

- Initial release
- Commands: test, user, tags, torrent (list/info), top (films/tv), wishlist
  (list/add/del), subtitles, rss-add, schema
- JSON/NDJSON output for easy piping to jq
- Configurable logging (level and destination)
- Schema introspection command
